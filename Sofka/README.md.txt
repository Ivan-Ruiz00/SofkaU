Naves Espaciales

Este programa permite consultar, crear y eliminar naves espaciales de 4 tipos diferentes: Lanzaderas, no tripuladas, no tripuladas sin peso, tripuladas.

Características

Consultar naves espaciales: El usuario puede buscar naves espaciales por tipo, nombre, país o destino.
Crear naves espaciales: El usuario puede crear naves espaciales de cualquiera de los 4 tipos disponibles.
Eliminar naves espaciales: El usuario puede eliminar naves espaciales existentes.

Requisitos

Java 8 o superior
JOptionPane

Uso

Ejecutar el archivo NavesEspaciales.java
Seleccionar la opción deseada: Consultar, Crear o Eliminar nave.
Seguir las instrucciones en pantalla para completar la acción deseada.

Notas

El programa guarda las naves creadas en un ArrayList para su posterior consulta.
Al momento de crear una nave se pide información como peso, empuje, estado, fechaDeLanzamiento, destino, combustible, país y nombre.
Al momento de consultar se pide el tipo de búsqueda y seguir las instrucciones en pantalla
Al momento de eliminar se pide el nombre de la nave a eliminar
El programa cuenta con un control de excepciones para evitar errores en la captura de información.
Se utilizan ventanas de diálogo para la captura de información y mostrar resultados.
